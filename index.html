Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Sistema de Portaria

    Sistema web desenvolvido para a empresa Alcoolvale, com funcionalidades como cadastro de tarefas, atribuição de responsáveis e acompanhamento do status das atividades.

  • Dashboard de Produção Desenvolvido em Python

    Dashboard de produção utilizando Python como back-end, conectado ao banco de dados Oracle para garantir a precisão e atualização das informações. O front-end foi construído utilizando JavaScript para oferecer uma experiência de usuário dinâmica e intuitiva. Esse Dashboard é especialmente voltado para o acompanhamento da safra, fornecendo uma visão detalhada da produção atual. Ele utiliza a conexão direta com o banco Oracle para trazer informações atualizadas em tempo real, permitindo que os usuários monitorem de perto o progresso da safra e tomem decisões estratégicas com base em dados precisos e atualizados. Além disso, o Dashboard oferece uma visualização em tempo real dos pontos de colheita em um mapa interativo. Essa funcionalidade permite que os usuários acompanhem a distribuição geográfica da colheita e identifiquem áreas de maior atividade. Com essa visualização espacial, os gestores podem tomar decisões mais informadas sobre alocação de recursos e logística, maximizando a eficiência da operação de colheita.

  • Dashboard Suprimento

    Esse e um Dashboard robusto utilizando Python, que se conecta diretamente ao banco de dados Oracle para fornecer uma visão abrangente de todas as solicitações de compra, status do estoque, valor atual do estoque e outras métricas relevantes. Essa ferramenta é essencial para facilitar as tomadas de decisões estratégicas, oferecendo insights detalhados e em tempo real sobre o suprimento da empresa. Com visualizações claras e intuitivas, o Dashboard proporciona uma análise abrangente e facilita a identificação de tendências e oportunidades de melhoria
